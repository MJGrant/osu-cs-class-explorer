About this project

OSU eCampus CS Course Explorer uses data collected from real students on the OSUOnlineCS subreddit. The data in this app is as useful as students like you make it. Add your own course review(s) via the survey here.

This app was developed by Mandi Grant, a current OSU student and professional web developer, using the Angular Fullstack Generator. Explore the project code on my GitHub and connect with me on LinkedIn. Want to add something? Send me a pull request!
