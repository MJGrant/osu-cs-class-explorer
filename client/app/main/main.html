About this project

OSU CS Course Explorer was developed for students of Oregon State University's online Computer Science program. Course descriptions are from OSU's official course list and the tips, difficulty, and time spent are scraped from this spreadsheet and are written by real students on the OSUOnlineCS subreddit. The course review data in this app is as useful as students like you make it. Add your own course review(s) via the survey here.

App developed by Mandi Grant, OSU CS student and professional web developer. Explore the project code on my GitHub and connect with me on LinkedIn.
